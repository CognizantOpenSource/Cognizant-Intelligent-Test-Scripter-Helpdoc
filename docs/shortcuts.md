# Shortcuts
---
## File Menu

| Shortcut Keys | Description |
| ------------ | ------------- | 
| Ctrl + Shift + N | Create a new project |
| Ctrl +Shift+ O | Open an existing project |
| Ctrl +Shift+ S | Save All |
| F1 | Open Help File |
